第1天: 6月28日 (周六) | 南京 → 宜昌

🚗 核心行程

  • **路线**: 南京浦口区出发 → G25宁合高速 → G42沪蓉高速 → 三峡大坝服务区 → 宜昌市区 。
  • **里程**: 约800-840公里，纯驾驶8.5-9.5小时 (行程紧张，建议调整) 。

🏞️ 核心观景

  • **三峡大坝坛子岭观景台** 。
  • **最佳光线**: 16:30-18:30，侧逆光 。
  • **备注**: 17:00后停止入场，需预约，电梯直达 。

🏨 住宿信息

  • **酒店**: 维也纳国际酒店(宜昌火车东站店) 。
  • **地址**: 伍家岗区桔乡路501号 。
  • **停车/充电**: 地下停车场，有2个7kW充电桩 。

🍲 餐饮推荐

  • **高速午餐 (六合服务区)**: 热干面、武汉豆皮 。
  • **晚餐 (宜昌)**: 长江颐居鱼庄 (招牌: 三峡活鱼锅) 。
  • **简餐**: 老街码头烧烤 (烤藕片、烤面筋) 。

第2天: 6月29日 (周日) | 宜昌 → 恩施

🚗 核心行程

  • **路线**: 宜昌上G50沪渝高速，直达恩施 。
  • **沿途景观**: 途径黄柏河峡谷 (G50, K1153) 。

🏞️ 核心观景

  • **云龙地缝观景道 (恩施大峡谷北门)**: 建议停留2小时，机位在第3悬挑平台 。
  • **恩施女儿城夜景**: 19:30-21:00灯火最佳 。

🏨 住宿信息

  • **酒店**: 恩施华美达安可酒店(土司城店)，连住4晚 。
  • **地址**: 市民之家民族路壹品首府 。
  • **停车/充电**: 地上停车场，有1个22kW充电桩 。

🍲 餐饮推荐

  • **高速午餐 (宣恩服务区附近)**: 土司柴火饭 (招牌: 柴火腊肉饭) 。
  • **晚餐 (恩施)**: 铛锅传奇 (招牌: 番茄土家腊肉火锅) 。

第3-5天: 6月30日-7月2日 | 恩施深度游

🚗 核心行程

  • **第3天**: 恩施大峡谷(七星寨、云龙地缝) 。
  • **第4天**: 平山草甸、恩施土司城 。
  • **第5天**: 屏山大峡谷漂流 。

🏞️ 核心观景

  • **七星寨玻璃栈道**: 7:30前限流 。
  • **屏山漂流终点**: U型岩壁。
  • **鹿院坪**: 瀑布星空 (20:30后需徒步2km) 。

🏨 住宿信息

  • **酒店**: 恩施华美达安可酒店(土司城店) 。
  • **洗衣**: 酒店有自助投币洗衣房，洗+干一体¥15/次 。

🍲 餐饮推荐

  • **七星寨午餐**: 观景木屋野菜面 (香椿鸡蛋面) 。
  • **屏山晚餐**: 鱼酥酸汤江团鱼 。
  • **夜宵**: 女儿城夜排档烤串，配土家米酒 。

🍲 餐饮推荐

  • **早餐/路餐 (六合服务区)**: 六合小笼包 。

打包与装备 🎒

核心着装策略

  • **高温城市日间**: 速干T恤、轻薄长袖防晒衫、亚麻长裤。
  • **山地早晚**: 抓绒内胆、软壳冲锋衣、速干长裤。
  • **雨天/漂流**: 轻量雨衣、涉水鞋、速干短裤，电子设备务必使用防水袋。

核心摄影装备

  • **机身/镜头**: Sony A1, 24-70mm F2.8 GM II, 70-200mm F2.8 GM II。
  • **无人机**: DJI套装 (电池×3)。
  • **数据备份**: 移动SSD 4TB ×2, 进行A/B备份。

车载与应急

  • **车载供电**: 300W逆变器, 100W USBC/GaN充电器, 65W无人机车充。
  • **车辆应急**: 备用充气泵+胎压表, 反光背心+三角警示架。

文化探索 🏛️

  • 三峡大坝: 长江治水千年夙愿的体现，被BBC评为“人类七大工程奇迹”之一 。
  • 恩施女儿城: 以土家族“女儿会”民俗为蓝本，如今复原了吊脚楼和摆手舞，夜里灯笼映河，号称“土家小凤凰” 。
  • 矮寨大桥: 世界首座跨度超1000米的钢桁梁悬索桥，桥面至峡谷垂高330米，号称“云端里的高速” 。
  • 漓江相公山: 典型的喀斯特峰丛地貌，是20元人民币背面图案的灵感来源 。
  • 武汉长江大桥: 新中国首座跨江铁路公路两用桥，1957年通车，桥墩包钢“水下焊接”技术为首创 。
