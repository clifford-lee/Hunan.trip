第1天: 6月28日 (周六) | 南京 → 宜昌

🚗 行程与路线

主题: 跑长途，看三峡大坝。

里程: 总里程约800-840公里，纯驾驶8.5-9.5小时。 强烈建议将此行程拆分为两天。

高德路线: 南京浦口区出发 → 宁合高速(G25) → 南京绕城高速(G2501) → 沪蓉高速(G42) → 三峡大坝服务区出口 → 宜昌市区出口。

时间安排: 06:15出发 ，根据下午抵达时间与体力状况，灵活决定是先入住酒店还是先游览大坝。

🏞️ 观景详解

核心景点: 三峡大坝坛子岭观景台。

最佳光线: 16:30-18:30，侧逆光能优美地勾勒出大坝线条。

注意事项: 需预约，17:00后停止入场 。电梯可直达，对长者友好。

文化背景: 三峡大坝是长江千年治水愿望的实现，被BBC评为“人类七大工程奇迹”之一。

🏨 住宿信息

酒店: 维也纳国际酒店(宜昌火车东站客运中心店)。

房型/价格: 高级双床房，约¥480。

地址/电话: 伍家岗区桔乡路501号 / 0717-6989999。

停车/充电: 地下免费停车，有2个7kW充电桩。

洗衣: 前台送洗，¥25/kg，次日取件。

🍲 餐饮推荐

高速午餐 (六合服务区): ●招牌: 热干面、武汉豆皮。

晚餐 (宜昌市区): ●长江颐居鱼庄 (夷陵大道东段88号)，招牌是“三峡活鱼锅”，本地人常自带米饭蘸汤。

简餐/夜宵: ○老街码头烧烤 (东站南广场对面)，推荐烤藕片、烤面筋。

👕 着装与装备

建议着装: 速干T恤 + 亚麻长裤。

关键装备: 折叠遮阳帽、偏光墨镜、冰袖。

第2天: 6月29日 (周日) | 宜昌 → 恩施

🚗 行程与路线

主题: 灵活安排，前往“鄂西林海”恩施。

路线: 离开宜昌上G50沪渝高速，直达恩施。

沿途景观: G50高速K1153处可远眺黄柏河峡谷。

🏞️ 观景详解

核心景点: 云龙地缝观景道、恩施女儿城夜景。

机位与技巧: 在大峡谷北门的第3悬挑平台可俯拍地缝全貌 ；女儿城夜景在19:30-21:00灯光最美。

🏨 住宿信息

酒店: 恩施华美达安可酒店(土司城店)，连住4晚。

房型/价格: 豪华家庭套房，约¥1023/晚。

地址/电话: 市民之家民族路壹品首府 / 0718-6025555。

停车/充电: 地上免费停车，有1个22kW充电桩。

🍲 餐饮推荐

高速午餐 (宣恩服务区附近): ●土司柴火饭 (宣恩高速出口500m)，招牌是柴火腊肉饭、土家酸汤鱼。

晚餐 (恩施女儿城): ●铛锅传奇 (土司城内)，招牌是番茄土家腊肉火锅，长者可选鸳鸯锅。

简餐: ○土家小卷粉 (城南美食街)，拌酸萝卜+小米椒。

👕 着装与装备

建议着装: 轻薄衬衫 + 防晒裤，以应对可能的午后阵雨。

关键装备: 雨衣需放在易取处，可备鞋套和车用空调毯。

第20天: 7月17日 (周四) | 武汉 → 南京

🚗 行程与路线

主题: 满载回忆，返回温馨的家。

路线: 汉阳区盛捷公寓出发 → G42沪蓉高速 → 六合服务区 → 南京。

里程: 约470公里，纯驾驶5.25小时。

时间安排: 08:15出发，预计16:15抵达南京。

🏞️ 观景详解

核心景点: 武汉长江大桥桥头堡。

文化背景: 新中国首座跨江大桥，毛主席亲笔题额，其“水下焊接”技术为首创，已列入世界纪念性建筑。

🏨 住宿信息

酒店: 今晚，睡在自己舒适的床上！

🍲 餐饮推荐

早餐/路餐 (六合服务区): ●六合小笼包。

午餐 (合肥服务区): ○臭鳜鱼。

👕 着装与装备

建议着装: 棉质T恤 + 弹力长裤。

关键装备: 准备一套备用换洗衣物，并记得补涂防晒霜。
